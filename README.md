# Restaurant Website

A complete restaurant website with menu management, shopping cart, billing, and payment features.

## Features

- **Menu Management (CRUD)**: Add, edit, and delete menu items in the Admin panel
- **Menu Display**: Beautiful grid layout showing all menu items with images
- **Shopping Cart**: Click on any menu item to add it to cart
- **Real-time Billing**: Automatic calculation of subtotal, tax (18%), and total
- **QR Code Payment**: Generate UPI QR code for payment
- **Bill Printing**: Print-friendly bill format
- **Clear Cart**: Remove all items from cart

## How to Use

1. **View Menu**: Open `index.html` in your browser
2. **Manage Menu**: Go to `admin.html` to add/edit/delete menu items
3. **View Cart**: Click on "Cart" in the navigation or go to `cart.html`
4. **Add Items**: Click on any menu item or use the "Add to Cart" button
5. **Checkout**: 
   - Click "Pay Now" to see QR code
   - Click "Print Bill" to print the bill
   - Click "Clear Cart" to remove all items

## Technical Details

- Pure HTML, CSS, and JavaScript
- Uses localStorage for data persistence
- Responsive design for mobile and desktop
- No backend required - works offline

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
